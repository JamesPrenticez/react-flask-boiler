# Backend
- Flask ``` pip3 install flask ```

## Install (linux)
``` sudo apt install python3-virtualenv ```
``` virtualenv -p python3.8 venv ```

## Run
0. cd backend
1. source venv/bin/activate
2. python3 app.py
3. http://localhost:5000/
