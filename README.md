# Reack / Flask Boilerplate

## Install (windows)
0. Install python and pip
1. Install/create virtual enviroment 
  - ``` pip install virtualenv ```
  - ``` virtualenv venve ```
2. Install packages
  - ``` pip3 install flask ```

## Run
0. cd frontend
1. . ../backend/venv/bin/activate
2. npm run dev
3. http://localhost:3000/
